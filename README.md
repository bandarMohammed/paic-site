# Programming and Artificial Intelligence Club Offers Website

# موقع عروض نادي البرمجة والذكاء الاصطناعي



موقع إلكتروني مخصص لنادي البرمجة والذكاء الاصطناعي (PAIC) في جامعة الطائف، يهدف إلى عرض الخصومات والعروض الخاصة للطلاب في المطاعم والمقاهي بطريقة سهلة ومنظمة مع خريطة تفاعلية لكل موقع.

### 🎯 الهدف

توفير منصة تجمع العروض الطلابية وربط الطلاب بالأنشطة التجارية المحلية.

### 🚀 المميزات

* عرض العروض والخصومات
* خريطة تفاعلية لكل موقع
* واجهة بسيطة وسهلة الاستخدام
* تنظيم واضح للمحتوى

### 🛠️ التقنيات

* HTML5
* CSS3
* JavaScript
* Google Maps Embed

---



A web-based platform for the Programming and Artificial Intelligence Club (PAIC) at Taif University. It showcases exclusive student discounts for restaurants and cafés with an interactive map and a clean, user-friendly interface.

### 🎯 Goal

To provide a centralized platform connecting students with local business offers.

### 🚀 Features

* Display offers and discounts
* Interactive map integration
* Clean and simple UI
* Organized layout

### 🛠️ Technologies

* HTML5
* CSS3
* JavaScript
* Google Maps Embed
